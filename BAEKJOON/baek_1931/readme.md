* 문제 해석 
조건에 맞는 회의의 최대 갯수를 구하려면 각 회의의 종료시각을 기준으로 정렬해야 한다.
여기서 2가지 경우를 고려해야한다.

 1. 종료시각이 서로 같은 두 회의
  이 경우를 해결하기 위해 종료시각이 서로 같은 회의끼리 다시 시작시각이 이른 순으로 정렬해야한다.
  
 2. 회의가 시작하자마자 종료하는 경우
  이 경우도 위에 방식으로 종료시각이 서로 같은 회의끼리 다시 시작시각이 이른 순으로 정렬하면 자연스럽게 해결가능.
